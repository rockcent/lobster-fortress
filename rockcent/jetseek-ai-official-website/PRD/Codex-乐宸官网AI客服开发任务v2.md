# 乐宸官网AI智能客服 - 详细开发任务

## 任务概述

在乐宸官网(rockcent.com)增加AI智能客服功能。

## 技术架构

### 1. 前端实现
- 在现有index.html中嵌入客服组件
- 悬浮按钮 + 对话窗口
- 纯前端实现，不依赖后端

### 2. 嵌入方式
```html
<!-- 直接在index.html中添加 -->
<script src="chat-widget.js"></script>
```

### 3. AI对话
- 使用Google Gemini API (免费额度)
- 预置知识库Prompt
